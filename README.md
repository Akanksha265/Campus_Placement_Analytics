# 🎓 Campus Placement Analytics Dashboard

> An end-to-end Data Analytics project that analyzes engineering student placement data using **SQL, Python, Excel, and Power BI** to uncover placement trends, recruiter insights, salary distributions, and key factors influencing student placements.

---

## 📌 Project Overview

This project demonstrates the complete data analytics workflow—from data cleaning and SQL analysis to Python-based exploratory data analysis (EDA) and interactive dashboard development in Power BI.

Using a synthetic dataset of **1,000 engineering students**, the project provides actionable insights into placement performance, recruiter hiring patterns, salary packages, internship impact, and academic performance.

---

## 🎯 Objectives

- Analyze engineering student placement trends
- Identify top recruiting companies
- Compare salary packages across branches
- Study the impact of internships on placements
- Analyze placement distribution by job role
- Examine the relationship between CGPA and salary package
- Build an interactive dashboard for data-driven decision-making

# 📌 Key Insights

- Computer Science recorded the highest placement rate.
- Higher CGPA generally corresponded to better salary packages.
- Students with internships achieved better placement outcomes.
- A few companies accounted for a significant share of campus recruitment.
- Software Engineering and Graduate Engineering roles were the most frequently offered positions.
- Salary packages varied noticeably across engineering branches.
